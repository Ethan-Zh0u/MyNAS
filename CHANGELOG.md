# 更新日志

本项目遵循语义化版本规则，所有可识别的发布版本都记录在这里。

## [0.3.0] - 2026-07-14

### 新增

- 上传继续使用 8 MB 分块，传输页面新增当前分块的浏览器级实时进度，已上传大小不再只按 `8 MB → 16 MB` 跳动。
- 新增 MyNAS 应用级读写字节计数；主页按真实采样间隔显示 `B/s`、`KB/s` 或 `MB/s`。
- 上传暂停、取消或断线时，界面回退到服务器已确认的分块边界，避免把尚未落到服务端的数据误认为已完成。

### 修复

- 修复硬盘详情保存旧采样对象，导致“旧值减新值”后读写速率长期显示为 0 的问题。
- 修复将两秒累计量显示成 `/2s` 而不是实际每秒速率的问题。
- 修复 Linux 页缓存延迟刷新物理块设备计数，导致上传期间绝大多数采样为 0 的问题；现在统计 MyNAS 实际接收和发送的数据。

### 验证

- 使用约 40 MB 文件完成真实 Tailscale 分块上传，文件成功校验并出现在 NAS 文件列表中。
- 连续 8 个 1 MB 探针分块中，服务端 `writeBytes` 每次准确增加 `1,048,576` 字节。
- 前端 15 项测试、TypeScript 检查、Go 测试、Windows/Linux ARM64 构建以及 Cloudflare Pages 发布均通过。

### 后续计划

- 增加账户退出与切换入口：清理当前 MyNAS 会话和设备状态，并支持切换 Tailscale 账号或未来的本地账号，避免用户只能手工清理浏览器状态。

## [0.2.0] - 2026-07-14

### 新增

- 新增多硬盘架构，分别展示每块硬盘的容量、已用空间、可用空间、在线状态和设备详情。
- 新增 `sudo mynas-setup` 树莓派接盘向导，支持 `ext4`、`NTFS3`、`exFAT`，自动排除系统盘并对格式化操作进行二次确认。
- 新增硬盘重命名，名称写入树莓派数据库并在重启后保留。
- 新增跨硬盘复制、移动、上传和按卷隔离的回收站。
- 新增多 MyNAS 设备管理，可添加、命名、重命名、切换和移除多台树莓派。
- 新增首次连接引导，覆盖 SSH 服务、电脑端终端位置、Tailscale 安装登录和多设备识别。
- 新增 Windows、macOS、Linux 操作系统切换和中文/英文界面。
- 新增主页深色主题开关、目录返回上级按钮和多硬盘演示模式。
- 新增面向零基础用户的接线与安装总指南，包含路由器网线、硬盘盒供电和硬盘连接安全说明。

### 改进

- 统一文件悬浮操作图标大小并重新设计命令复制按钮。
- 已配对设备后续访问不再重复显示首次安装教程；离线时显示恢复连接步骤。
- 上传任务支持暂停、继续、取消和离开页面后的状态恢复。
- 配置文件刷新不再覆盖用户自定义的硬盘名称。

### 当前限制

- 通用的一键树莓派初装程序尚未完成；当前生产部署脚本仍面向项目维护者的测试设备。
- 远程访问和身份认证仍以 Tailscale 为默认安全边界。

## [0.1.0] - 2026-07-13

### 已有功能

- 提供 React + Go 的个人 NAS 管理界面。
- 支持文件浏览、上传、下载和基础文件操作。
- 提供磁盘容量、读写状态和审计记录。
- 使用 Tailscale Serve 作为当前远程访问和身份认证边界。
- 提供 Windows 本地开发、Linux ARM64 构建和树莓派部署脚本。

### 已知问题

- 首页文件夹的快速操作按钮尺寸不一致。
- 部署配置仍与当前树莓派、挂载路径和 Tailscale 环境绑定。

### 后续计划

- 通用树莓派与外接硬盘快速部署。
- 视频在线预览。
- 可选的本地账号密码登录。
